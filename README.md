# Pokedex API
Proxies PokeAPI and provides manipulated images and data for Pokedex

## Setup
All environments, create an `assets/images` folder

### Dev setup
1. `yarn install`
2. `yarn start:dev`

### Server setup
1. `yarn install`
2. `yarn build`
3. `yarn start`

### Docker
1. `yarn build`
2. `docker-compose up`
